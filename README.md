# HUDTweaks
**Mod for Spin Rhythm XD that adds more options and tweaks to the gameplay HUD**

## Additions
- Visibility toggles for HUD elements that do not have one:
  - Combo 
  - Health bar
  - Side flashing when hurt
  - Multiplier bar
  - Multiplier text
  - Score
  - Track highlights/track strip
  - Wheel grips
  - Hiding accuracy log rows for timings above a set threshold
- Augmented info displays:
  - The score label can be changed to show current accuracy and Perfect+ counts
  - The health label can be changed to show the current *precise* health value
  - Time can be showed in beats, rather than minutes and seconds
  - Track information can be changed using custom formatting tags
- Colors:
  - Bold numbers for combo and score values
  - Multiplier elements
  - Standard HUD text
  - Time elements
  - Health elements
  - PFC/FC indicators

## Dependencies
- SpinCore
